# Mood-DJ
Song suggestion bot based on user mood.

## Dependencies:
[Pandas](https://github.com/pandas-dev/pandas)

[Flair](https://github.com/zalandoresearch/flair)


## Usage

#### Create Model

```
$ python create_model.py
```

#### Run

```
$ python chat.py best-model.pt
```
